# Template Hedera Agent Kit Next.js

## Getting Started

Required env variables:
```
OPENAI_API_KEY=""
HEDERA_ACCOUNT_ID=""
HEDERA_PRIVATE_KEY=""
HEDERA_NETWORK=""

NEXT_PUBLIC_WALLET_CONNECT_ID=""
```


run the server:

1. `npm install`
2. `npm run dev`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
